# CodeAlpha Tasks – Data Analytics

This repository contains my completed Data Analytics internship projects from CodeAlpha, focusing on Web Scraping and Exploratory Data Analysis (EDA) using Python.

---

## Projects Overview

### 1. Web Scraping – Job Listings

* Built a scraper using BeautifulSoup
* Extracted job data (title, company, location, URL)
* Stored structured data in CSV format

Key Features:

* Automated data collection
* Clean and structured dataset generation
* Timestamp tracking for scraped data

---

### 2. Exploratory Data Analysis (EDA)

* Performed data cleaning and preprocessing
* Analyzed job trends and patterns
* Built a complete dashboard using Matplotlib

Key Insights:

* Most job roles appear only once (high uniqueness)
* Python-related roles appear frequently
* No null values or duplicates found
* Dataset lacks time variation (single scrape)
